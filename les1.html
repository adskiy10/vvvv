CSS Селектори

CSS селектори використовуються для "пошуку" (або вибору) HTML-елементів, які ви хочете стилізувати.

Види селекторів:

1. CSS селектор елемента

Селектор елементів обирає HTML елементи на основі імені елемента.

Приклад:

Тут всі елементи <p> на сторінці будуть вирівняні по центру з червоним кольором тексту:

p {
  text-align: center;
  color: red;
}

2. CSS селектор id

Селектор id використовує атрибут #id HTML елемента для вибору визначеного елемента.

Ідентифікатор #id елементу є унікальним на сторінці, тому селектор #id використовується для вибору одного унікального елемента!

Щоб обрати елемент з визначеним ідентифікатором, напишіть символ хешу (#), а потім id елемента.

Приклад:

Наведене нижче правило CSS, яке буде застосовано до елементу HTML з id="para1":

#para1 {
  text-align: center;
  color: red;
}

Примітка: Ім'я Id не може починатися з цифри!

3. CSS селектор class

Селектор class обирає HTML елементи з визначеним атрибутом класу.

Щоб обрати елементи з визначеним класом, введіть символ крапки (.), а потім ім'я класу.

Приклад:

В цьому прикладі всі HTML елементи з class="center" будуть червоними та вирівняні по центру:

.center {
  text-align: center;
  color: red;
}

Ви також можете вказати, що клас має зачіпати лише визначені HTML елементи.

Приклад:

В цьому прикладі лише елементи <p> з class="center" будуть вирівняні по центру:

p.center {
  text-align: center;
  color: red;
}

HTML елементи також можуть посилатись на кілька класів:

Приклад:

В цьому прикладі елемент <p> буде стилізовано у відповідності з class="center" та class="large":

<p class="center large"> Цей параграф реферер для двох класів.</p>

Примітка: Ім'я класу не може починатися з цифри!

4. Універсальний CSS селектор

Універсальний селектор (*) виділяє всі елементи HTML на сторінці.

Приклад:

Наведене нижче правило CSS впливає на кожен HTML елемент на сторінці:

* {
  text-align: center;
  color: blue;
}

5. CSS Селектор групування

Селектор групування обирає всі HTML елементи з однаковими визначеннями стилю.

В наступному коді CSS елементи h1, h2 та p мають однакові визначення стилю:

h1 {
  text-align: center;
  color: red;
}

h2 {
 text-align: center;
  color: red;
}

p {
  text-align: center;
  color: red;
}

Буде краще згрупувати селектори, щоб мінімізувати код.

Щоб згрупувати селектори, розділяйте кожен селектор комою.

Приклад:

В цьому прикладі згрупували селектори із коду вище: 

h1, h2, p {
  text-align: center;
  color: red;
}
